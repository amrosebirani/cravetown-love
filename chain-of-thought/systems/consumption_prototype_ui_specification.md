## 1. Overview

### Purpose
This is an **interactive testing prototype** for the consumption system. The player can:
- Manually inject commodities into town inventory
- Create and modify characters (class, traits, satisfaction)
- Observe allocation behavior in real-time
- Trigger consequences (riots, emigration)
- View detailed analytics and craving heatmaps

### Core Philosophy
- **Maximum Visibility**: All internal state exposed
- **Maximum Control**: Every parameter adjustable
- **Real-time Feedback**: Changes immediately visible
- **Testing Focus**: Built for rapid iteration, not polish

- Hover: Show tooltip with quick stats

**Resource Button Click:**
- [+1]: Add 1 unit
- [+5]: Add 5 units
- [+10]: Add 10 units
- [Set]: Open input field for custom value

**Bar Chart Hover:**
- Show exact numerical value
